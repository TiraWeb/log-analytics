Log Analytics Platform

Statistical anomaly detection pipeline for microservice log data at scale. Uses Z-score profiling over 5-minute aggregation windows to detect error rate spikes, latency regressions and infrastructure saturation events. Cross-correlates log-derived signals with CloudWatch infra metrics for root cause diagnosis.
